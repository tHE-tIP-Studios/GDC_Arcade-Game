# GDC_Arcade-Game
Fourth Game Design Club Game

# Player Control  Scheme

WARNING!
'There are no diagonals, just 4 directions' 

## P1

- Up - `W`

- Down - `S`

- Left - `A`

- Right - `D`

- Button 1 - `X`

- Button 2 - `C`

- Button 3 - `V`

## Player 1

- Up - `↑`

- Down - `↓`

- Left - `←`

- Right - `→`

- Button 1 - `J`

- Button 2 - `K`

- Button 3 - `L`
